# Professional Portfolio Website

A stunning, modern, and highly interactive professional portfolio website built with cutting-edge technologies. This website is designed to showcase skills, experience, and projects with a premium aesthetic and smooth user experience.

## ✨ Features

- **Dynamic Hero Section**: A captivating first impression with smooth animations and clear call-to-actions.
- **Interactive About Me**: Detailed background and introduction using premium typography and glassmorphism.
- **Visual Skills Showcase**: An engaging way to display professional expertise using modern UI components.
- **Project Portfolio**: A filterable or categorizable showcase of work with detailed descriptions and links.
- **Contact Form**: A sleek, functional contact section for inquiries and networking.
- **Premium Aesthetics**: Dark mode by default, vibrant gradients, and smooth micro-animations.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.

## 🚀 Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **UI Components**: [Radix UI](https://www.radix-ui.com/), [Material UI (MUI)](https://mui.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Typography**: Modern fonts (Inter, Roboto)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sohagabir112/ProfessionalPortfolioWebsite.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ProfessionalPortfolioWebsite
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

To create a production build:
```bash
npm run build
```
